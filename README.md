# LOS - Lodi Operating System

**LOS** (Lodi Operating System) is a remastered Linux Mint-based OS designed specifically for **software development**. It offers a flexible environment where users can choose their development focus during the first install, such as **Web Development, Python Development, or Data Science**.

---

## **Key Features**
- Flexible development environments tailored for different focuses
- First-install setup allows you to choose your development path
- Built-in tools and utilities for programming, scripting, and testing
- Pre-installed themes, icons, and wallpapers for a clean workspace
- Documentation and guidance available through GitHub

---

## **Installation**
1. Download the latest LOS ISO from [Lodi7/LOS releases](https://github.com/Lodi7/LOS/releases)
2. Create a bootable USB using **Rufus** (Windows) or **balenaEtcher** (Linux/macOS)
3. Boot from the USB drive
4. Follow the setup wizard and choose your development environment
5. Start using LOS with your selected tools and environment
